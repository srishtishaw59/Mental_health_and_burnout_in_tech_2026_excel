# Mental_health_and_burnout_in_tech_2026_excel




## Project Overview

This project is an Excel dashboard that explores mental health and burnout among tech employees. The dashboard helps understand how factors such as work hours, sleep, exercise, therapy, and work mode affect employee well-being.

## Objectives

* Analyze employee burnout and stress levels.
* Study the impact of work-life balance.
* Compare burnout across job roles, industries, and countries.
* Find patterns that can help improve employee mental health.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Design

## Key Insights

* Longer work hours are linked to higher stress and burnout.
* Employees who sleep less tend to have higher stress levels.
* Remote workers generally have better work-life balance.
* Regular exercise helps reduce stress.
* High burnout is often connected with lower job satisfaction.

## Dashboard Features

* Burnout Score Analysis
* Stress Score Analysis
* Work-Life Balance Comparison
* Country-wise Burnout Analysis
* Therapy Usage Analysis
* Salary and Job Role Analysis
* Interactive Filters and Slicers

## Conclusion

This project shows that employee well-being is affected by work pressure, sleep habits, and work-life balance. Healthy work practices, exercise, and mental health support can help reduce burnout and improve productivity.
